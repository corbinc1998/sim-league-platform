# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for sim-league-platform.

An ADR documents a single significant decision: the context that prompted it, the decision itself, and the consequences. Each ADR is immutable once accepted — superseded decisions are documented in new ADRs that reference the old ones.

Format follows [Michael Nygard's template](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions).
